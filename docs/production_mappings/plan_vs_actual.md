# Plan vs Actual Mapping

**Full Forms**:
- EMS = Electronic Manufacturing Services
- PDI = Pre-Dispatch Inspection

**Simple Explanation**: Compare actual production to planned. ERPNext uses Job Cards for tracking.

## AS-IS Steps (from AS-IS_Process.xlsx)
1. Capture daily production data at EMS.
2. Include PDI, packing, rejections.
3. Compare to planned.

## ERPNext Implementation Steps
1. Use Job Cards for production tracking.
2. Generate plan vs actual reports.
3. Track rejections and variances.

## Pain Points (AS-IS)
- Manual daily tracking.

## ERPNext Solutions
- Automated tracking and reports.

# Handover Process Mapping

**Full Forms**:
- PFD = Process Flow Diagram (workflow document)

**Simple Explanation**: R&D hands over completed designs and documents to Production for mass manufacturing. ERPNext tracks this in Projects module.

## AS-IS Steps (from AS-IS_Process.xlsx)
1. Submit requisite documents to Production Team.

## ERPNext Implementation Steps
1. Update Project in Projects module with handover milestone.
2. Attach PFD and other documents.
3. Notify Production team.
4. Mark handover complete.

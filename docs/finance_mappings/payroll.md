# Payroll Mapping

**Full Forms**:
- LOP = Loss of Pay
- HR = Human Resources

**Simple Explanation**: Process monthly salary calculations and disbursements. ERPNext Payroll module handles this.

## AS-IS Steps (from AS-IS_Process.xlsx)
1. Input received from HR related to LOP and any other adjustments.
2. We create the payroll sheet based on the information received from HR thru keka.

## ERPNext Implementation Steps
Note: Payroll is out of scope per sheet feedback, but if implemented:
1. HR enters LOP and adjustments in ERPNext.
2. System calculates salaries using Payroll entries.
3. Generate payslips and process payments.

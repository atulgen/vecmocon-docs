# AR Ageing and AP Ageing Mapping

**Full Forms**:
- AR = Accounts Receivable
- AP = Accounts Payable

**Simple Explanation**: Generate ageing reports for receivables and payables. ERPNext ageing reports.

## AS-IS Steps (from AS-IS_Process.xlsx)
AR:
1. Export the AR report from Tally.
2. Create the aging for overdue/outstanding.
3. Follow Up with the customer thru mail.

AP:
1. Export the AP report from Tally.
2. Create the ageing for overdue/outstanding.
3. Payment is done based on weekly sheet filled by purchase team.

## ERPNext Implementation Steps
1. Generate AR/AP ageing reports.
2. Set up alerts for overdue items.
3. Automate follow-ups.

## Pain Points (AS-IS)
- Manual follow-ups.

## ERPNext Solutions
- Automated ageing and alerts.

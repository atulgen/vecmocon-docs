# Admin Payments Process Mapping

**Full Forms**:
- None

**Simple Explanation**: Process payments for administrative goods and services.

## AS-IS Steps (from AS-IS_Process.xlsx)
1. Admin team identifies payment need (e.g., supplies, services)
2. Collect and check invoice/supporting documents
3. Prepare payment request
4. Initiate approval chain via email
5. Upon approval, send request and documents to Finance
6. Finance processes payment
7. Confirmation of payment

## ERPNext Implementation Steps
1. Admin submits payment request
2. Approval workflow
3. Finance creates Payment Entry
4. Process payment
5. Confirm

## Pain Points (AS-IS)
- Missing documents

## ERPNext Solutions
